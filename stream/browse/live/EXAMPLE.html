Live Stream Player - Usage Examples

1. Basic HLS Stream

Link to a live HLS stream:

live/index.html?src=https://example.com/live/stream.m3u8&title=My Live Channel

Try it

2. Supabase Storage HLS Stream

Link to a stream stored in Supabase:

live/index.html?src=https://your-project.supabase.co/storage/v1/object/public/stream/live/channel.m3u8&title=Supabase Channel

3. With Subtitles

Add subtitles to your live stream:

live/index.html?src=https://example.com/live/stream.m3u8&title=News Channel&subtitles=https://example.com/subtitles.vtt

4. HTML Link Example

How to create a link in your browse page:

<a href="live/index.html?src=https://example.com/live/stream.m3u8&title=Channel 1">
  Watch Channel 1 Live
</a>

5. JavaScript Dynamic Link

Create links dynamically:

const channel = {
  name: 'News Channel',
  url: 'https://example.com/live/news.m3u8'
};

const link = document.createElement('a');
link.href = `live/index.html?src=${encodeURIComponent(channel.url)}&title=${encodeURIComponent(channel.name)}`;
link.textContent = `Watch ${channel.name}`;

Stream Input Types Supported:

  • HLS/M3U8 - Recommended for live streams (adaptive bitrate, low latency)
  • MP4/WebM - Regular video files (not ideal for live, but supported)

Getting Your Stream URL:

  1. Set up a streaming service (AWS MediaLive, Wowza, OBS, etc.)
  2. Configure it to output HLS format (.m3u8)
  3. Get the public URL to your M3U8 playlist file
  4. Use that URL in the src parameter
